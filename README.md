# SDII to WAV (IOSE Utility)

**"There is no such thing as unreadable data. Only data we don't yet know how to read."**
（読めないデータは存在しない。読み方がわからないデータがあるだけ。）

This is a browser-based audio rescue tool developed as part of the **IOSE (Intelligent Operating System Environment)** project. It is designed to rescue digital heritage that has become inaccessible due to the barriers of legacy operating systems and proprietary formats.

## 💿 Philosophy & Background
"Sound Designer II (SDII)" was once a standard audio format in professional music production. However, due to the evolution of operating systems and changes in file systems (such as the loss of Mac resource forks), these files have become "unopenable" in most modern environments.

While the digital world has grown more convenient, we face a "Digital Tower of Babel" problem: when we lose the "metadata key" (the OS or specific format reader), the pure waveform data inside remains intact, yet entirely out of reach.

This tool bypasses the overprotective filters of modern OS layers. It directly reads the raw binary data of the SDII waveform and reconstructs it into the modern universal language: the WAV format.

## ✨ Features
* **100% Browser-Based:** No server uploads. The entire process runs safely and instantly within your local browser environment.
* **Batch Processing:** Simply drag and drop multiple SDII files at once to instantly convert, preview, and download them as WAV files.
* **Environment Independent:** Running on a single HTML file, it works across any OS and will continue to function semi-permanently, even offline.

## 🚀 How to Use
1. Open the tool's URL.
2. Drag and drop your SDII files into the drop zone (multiple files supported).
3. The waveforms will be loaded instantly, allowing you to preview the audio directly in the browser.
4. Click the "Download WAV" button to save your rescued data.

## 🌐 Hope for the Future: The Next Rescue
"SDII to WAV" is merely the first step in reclaiming our lost data.

From proprietary sampler formats to old hardware save data, countless digital files around the world have been abandoned simply because they can no longer be opened. The **IOSE (Intelligent Operating System Environment)** project will continue to dissect the architecture of these forgotten formats, striving to resurrect them into universal forms. 

We hope that the precious session data and artworks of the past will once again resonate through modern speakers.

# SDII to WAV (IOSE Utility)

**「読めないデータは存在しない。読み方がわからないデータがあるだけ。」**

これは、過去のOSや独自フォーマットの壁に阻まれ、開けなくなってしまったデジタル遺産を救出するプロジェクト「IOSE（Intelligent Operating System Environment）」の一環として開発された、ブラウザ完結型のオーディオレスキューツールです。

## 💿 開発の意義とIOSEの哲学
かつて音楽制作の現場で標準的だった「Sound Designer II (SDII)」フォーマット。しかし、OSの進化やファイルシステムの変更（リソースフォークの消失など）により、現在では多くの環境でただの「開けないファイル」となってしまっています。

デジタルの世界は便利になった一方で、OSやフォーマットという「メタデータの鍵」を失うと、中身の波形データ自体は無傷でもアクセスできなくなるという「デジタルのバベルの塔」のような問題を抱えています。

本ツールは、OS層のお節介なフィルターをバイパスし、純粋なバイナリデータとしてSDIIの波形を直接読み込み、現代の共通言語である「WAVフォーマット」へと再構築します。

## ✨ 特徴
* **完全ブラウザ完結:** サーバーへのアップロードは一切行いません。あなたのブラウザ（ローカル環境）内だけで安全かつ瞬時に処理が完結します。
* **一括バッチ処理:** 複数のSDIIファイルをまとめてドラッグ＆ドロップするだけで、一気にWAVへと変換・試聴・ダウンロードが可能です。
* **環境非依存:** HTMLファイル1つで動作するため、OSを問わず、オフライン環境でも半永久的に動作し続けます。

## 🚀 使い方
1. 本ツールのURLを開きます。
2. 画面中央のドロップゾーンに、変換したいSDIIファイルをドラッグ＆ドロップします（複数選択可）。
3. 瞬時に波形が読み込まれ、ブラウザ上で試聴が可能になります。
4. 「WAVをダウンロード」ボタンを押して、救出されたデータを保存してください。
